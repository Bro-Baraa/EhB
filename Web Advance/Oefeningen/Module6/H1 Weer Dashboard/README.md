# H1 - Weer dashboard

Deze oefening is een eenvoudige weer-app.
Je vult een stad in en de app zoekt de locatie en het weer op.
Daarna zie je temperatuur, beschrijving, luchtvochtigheid, wind, zonsopgang en zonsondergang.
Je kan ook wisselen tussen Celsius en Fahrenheit en steden als favoriet gebruiken.



# H1 - Weather Dashboard

This exercise is a simple weather app.
You enter a city and the app looks up the location and the weather.

Afterwards, you see the temperature, description, humidity, wind, sunrise, and sunset.

You can also switch between Celsius and Fahrenheit and mark cities as favorites.

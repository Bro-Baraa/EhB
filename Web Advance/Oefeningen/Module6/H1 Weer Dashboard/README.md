# H1 - Weer dashboard

Deze oefening is een eenvoudige weer-app.
Je vult een stad in en de app zoekt de locatie en het weer op.
Daarna zie je temperatuur, beschrijving, luchtvochtigheid, wind, zonsopgang en zonsondergang.
Je kan ook wisselen tussen Celsius en Fahrenheit en steden als favoriet gebruiken.

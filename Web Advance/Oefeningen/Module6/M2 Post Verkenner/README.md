# M2 - Post verkenner

Deze oefening haalt posts op van JSONPlaceholder.
Je kan zoeken op titel, sorteren en kiezen hoeveel posts je wil zien.
Elke post toont de titel, een kort stukje tekst, de post-ID en de gebruiker-ID.



# M2 - Post Explorer

This exercise retrieves posts from JSONPlaceholder.

You can search by title, sort, and choose how many posts you want to see.
Each post displays the title, a short piece of text, the post ID, and the user ID.

# H1 - Mini Learning Platform

This project is a small learning platform made with HTML, CSS, and JavaScript.

## Features

- Add courses with a title and description
- Each course gets default modules
- Enroll students in a course
- Add scores for students
- Generate a simple student report
- Remove courses
- Update dropdown menus when data changes

## JavaScript Concepts

- const
- arrays
- objects
- push()
- splice()
- for...of loop
- template literals
- DOM selection
- event listeners
- innerHTML

## Files

- index.html
- style.css
- script.js
- README.md

## How to Run

Open `index.html` with Live Server in VS Code.

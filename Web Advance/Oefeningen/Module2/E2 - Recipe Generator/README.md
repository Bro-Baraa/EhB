# E2 - Recipe Generator

This project generates a simple recipe card with JavaScript.

## Features

- Enter a recipe name
- Enter preparation time
- Add ingredients line by line
- Generate a recipe card
- Show ingredients in a list

## JavaScript Concepts

- template literals
- split()
- arrays
- for...of loop
- DOM selection
- event listener
- innerHTML

## Files

- index.html
- style.css
- script.js
- README.md

## How to Run

Open `index.html` with Live Server in VS Code.

# H1 - Dynamic Menu Builder

This project creates a dynamic navigation menu with JavaScript.

## Features

- Automatically creates a menu from h1 and h2 tags
- Main chapters and sub sections
- Clickable menu links
- Smooth scrolling to sections
- Active menu item
- Expand and collapse submenu

## Technologies

- HTML
- CSS
- JavaScript

## Files

- index.html
- style.css
- script.js

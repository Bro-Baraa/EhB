# E2 - Loading Message

This project is a simple loading screen made with HTML, CSS, and JavaScript.

## Features

- Shows a "Loading..." message while the page is loading
- Changes the text to "Welcome!" after the page is fully loaded
- Hides the message automatically after 2 seconds

## Technologies

- HTML
- CSS
- JavaScript

## JavaScript Concepts Used

- DOM selection
- Event listeners
- window load event
- setTimeout()
- textContent
- style manipulation

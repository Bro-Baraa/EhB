# H1 - Asynchronous Light Show

This exercise creates a small light show.

When you click start, the blocks light up one by one.

This happens with `async`, `await`, and a short waiting time.





# H1 - Asynchrone Lichtshow

Deze oefening maakt een kleine lichtshow.

Als je op start klikt, lichten de blokken één voor één op.

Dit gebeurt met `async`, `await` en een korte wachttijd.
